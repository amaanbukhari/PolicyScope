# Utilities

This folder contains shared utility helpers for PolicyScope.

## Current Status

At the current stage of the project, utility usage is still lightweight. This folder exists to support shared helper functions that are better kept separate from the main feature-specific modules.

## Files

- **normalize.js**  
  Intended to store reusable normalization helpers for text cleanup, formatting consistency, and other shared utility behavior that may be needed across parsing, detection, or popup logic.

## Purpose

The purpose of this folder is to avoid repeating low-level helper logic across multiple parts of the project. As PolicyScope continues to grow, this folder can be expanded to hold more shared utility functions that support cleaner and more maintainable code.
